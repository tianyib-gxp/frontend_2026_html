Session 1: 最初のHTMLを表示する

まずはタグを書いて、ブラウザ表示が変わる体験を作ります。

ウェブの3層構造

ウェブページは3つの技術で成り立っています。
HTML：構造と意味　／　CSS：見た目　／　JavaScript：動作と対話性

技術 役割 例
HTML 構造・意味付け 見出し、段落、リスト
CSS 見た目・スタイル 色、フォント、レイアウト
JavaScript 動作・対話性 ボタンのクリック、DOM操作

説明

HTMLは「見た目」より先に「意味のある構造」を作る言語です。この回では、見出し・段落・リストを使って自己紹介ページの骨組みを作ります。

やってほしいこと

  1. h1に自己紹介タイトルを書く
  2. pを2つ追加して、自己紹介文を書く
  3. ulとliで、好きなことを箇条書きする
[ポイント]
  • タグは役割を持つ。見出しはh1、文章はp、一覧はulとliを使う
  • HTMLを書き換えたら、ブラウザをリロードして変化を確認する
  • 最初はタグを試しながら慣れることが大切
