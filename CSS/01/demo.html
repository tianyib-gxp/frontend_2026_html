1-1. width / height

px は固定サイズ、% は親要素に対する割合です。

px 指定（固定サイズ）

ブラウザ幅を変えても大きさが変わりません。

width: 200px
height: 80px
width: 350px
height: 80px
width: 100px
height: 150px
.box-200px { width: 200px; height: 80px; }
.box-350px { width: 350px; height: 80px; }
.box-100px { width: 100px; height: 150px; }

% 指定（親要素に対する割合）

ブラウザ幅を変えると一緒に伸縮します。

width: 100%
width: 75%
width: 50%
width: 25%
.box-100pct { width: 100%; }
.box-75pct  { width: 75%;  }
.box-50pct  { width: 50%;  }
.box-25pct  { width: 25%;  }
