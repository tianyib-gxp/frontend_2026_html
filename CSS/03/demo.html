セレクター — id / class / タグ

CSS でどの要素にスタイルを当てるかを「セレクター」で指定します。

タグセレクター（要素セレクター）

HTML タグ名をそのまま書くと、そのタグ全てにスタイルが当たります。

この段落は <p> タグなので青になります。

こちらも <p> タグなので同じスタイルです。

この <span> は別のタグなので影響されません。 
/* タグセレクター */
p {
  color: #1d3461;
  font-weight: bold;
}

クラスセレクター（.クラス名）

HTML に class="名前" を付けて、CSS で .名前 と書きます。
同じクラスは複数の要素に使い回せます。

class="highlight" が付いた段落（黄色背景）

クラスなし — スタイルが当たりません

こちらも class="highlight" なので同じスタイル

div タグでも同じクラスが使えます
/* クラスセレクター */
.highlight {
  background-color: #fff3bf;
  padding: 8px 12px;
  border-left: 4px solid #f4a261;
  border-radius: 4px;
}

id セレクター（#id名）

HTML に id="名前" を付けて、CSS で #名前 と書きます。
id は1ページに1つだけ使うのがルールです（クラスと違って使い回し不可）。

id="main-title" が付いた要素（特別なスタイル）

id なし — スタイルなし

id なし — スタイルなし

/* id セレクター */
#main-title {
  font-size: 24px;
  color: #9b2335;
  font-weight: bold;
  border-bottom: 2px solid #9b2335;
  padding-bottom: 8px;
}

class と id の使い分け

class id
HTML の書き方 class="名前" id="名前"
CSS の書き方 .名前 { ... } #名前 { ... }
同じページで複数使える？ ○ 何度でも使える ✕ 1ページに1つだけ
1つの要素に複数指定？ ○ スペース区切りで複数OK ✕ 1要素に1つだけ
主な用途 繰り返すスタイル（ボタン・カードなど） ページ上で唯一の要素（ヘッダーIDなど）

複数クラスの組み合わせ

1つの要素に複数のクラスをスペース区切りで付けられます。スタイルが重なり合います。

box のみ
box + red
box + large
box + red + large（全部合わせる）
.box  { padding: 8px 16px; border-radius: 4px; border: 2px solid #ccc; }
.red  { background: #e63946; color: #fff; border-color: #e63946; }
.large { font-size: 20px; padding: 16px 24px; }

子孫セレクター（スペースで繋ぐ）

「.親 子」のようにスペースで繋ぐと、特定の親の中にある子要素だけに当たります。

カードの中の <p> — 青色

カードの中の別の <p> — こちらも青色

カードの外の <p> — スタイルなし

/* .card-demo の中の p だけに当たる */
.card-demo p {
  color: #1d3461;
  font-weight: bold;
}
