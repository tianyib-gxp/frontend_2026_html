1-2. margin / padding

margin は要素の「外側」、padding は要素の「内側」の余白です。

margin — 要素の外側の余白

ボックスと周囲の要素との間隔を作ります。背景色は margin には適用されません。

margin: 24px
margin: 24px
margin: 24px
.box { margin: 24px; }

padding — 要素の内側の余白

ボックスの中身（テキストなど）とボーダーの間に余白を作ります。背景色は padding にも適用されます。

padding: 24px
テキストとボーダーの間の余白
.box { padding: 24px; }

box-sizing: border-box（推奨）

padding を加えても要素の width が変わらないようにします。

box-sizing: content-box（デフォルト）
width: 200px + padding: 20px → 実際の幅 = 240px

幅が広がる！

box-sizing: border-box（推奨）
width: 200px + padding: 20px → 実際の幅 = 200px のまま

幅が保たれる
