2-1. display

block / inline / inline-block の違いを確認しましょう。

display: block

横幅いっぱいに広がり、前後に改行が入ります。width / height / margin / padding が自由に使えます。

ブロック要素 A ブロック要素 B ブロック要素 C 
.item { display: block; width: 200px; }

display: inline

テキストと同じように横に並びます。width / height は無視されます。上下の margin も効きません。

インライン A インライン B インライン C インライン D 
.item { display: inline; }
/* width / height を指定しても効かない */

display: inline-block

横に並びながら（inline の性質）、width / height / margin が使えます（block の性質）。ナビゲーションメニューのボタンなどに使います。

ボタン A ボタン B ボタン C 
.item {
  display: inline-block;
  width: 100px;
  height: 40px;
  margin: 4px;
}
