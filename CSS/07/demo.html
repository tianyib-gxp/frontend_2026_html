3. テキストスタイル

font-size / font-weight / font-family / color / text-align

font-size（文字の大きさ）

font-size: 12px — 小さいテキスト

font-size: 16px — 本文サイズ（基本）

font-size: 24px — 見出しサイズ

font-size: 36px — 大見出し

font-weight（文字の太さ）

font-weight: 300 — 細い

font-weight: 400 — 普通（normal）

font-weight: 700 — 太い（bold）

font-weight: 900 — 最大の太さ

font-family（書体）

font-family: sans-serif — ゴシック体（読みやすい）

font-family: serif — 明朝体（フォーマル）

font-family: monospace — 等幅（コード用）

color（文字色）

color: tomato — 色名で指定

color: #1d3557 — 16進数で指定

color: rgb(76, 201, 240) — RGB で指定

color: inherit — 親要素の色を引き継ぐ

text-align（文字の水平位置）

text-align: left — 左揃え（デフォルト）

text-align: center — 中央揃え

text-align: right — 右揃え
