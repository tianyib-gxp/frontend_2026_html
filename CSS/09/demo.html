5-1. max-width / min-width

ブラウザ幅を変えて、各ボックスの挙動を確認しましょう。

max-width — 最大幅を制限する

幅を広げすぎないようにします。レスポンシブレイアウトで本文コンテンツを読みやすく保つのに使います。

max-width: 600px
幅を広げても600pxを超えない
max-width: 400px
幅を広げても400pxを超えない
.container {
  width: 100%;       /* 基本は100%（画面幅に合わせる） */
  max-width: 600px;  /* でも600px以上には広がらない */
  margin: 0 auto;    /* 中央寄せ */
}

min-width — 最小幅を確保する

幅を縮めすぎないようにします。ボタンやカードが小さくなりすぎるのを防ぎます。

min-width: 150px
150px未満にならない
min-width: 200px
200px未満にならない
.button {
  min-width: 120px;  /* ラベルが短くてもボタンが小さくならない */
  padding: 8px 16px;
}

レスポンシブの基本パターン

このページ自体が max-width の例です。ブラウザ幅を広げても本文エリアは広がりません。

このコンテナは width: 100% かつ max-width: 800px です。
幅を広げてみてください。800px以上は広がりません。
