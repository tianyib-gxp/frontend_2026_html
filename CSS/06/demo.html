2-3. z-index

重なり順（前後）を数値で制御します。大きいほど手前に表示されます。

基本：2つの要素を重ねる

z-index の数値が大きい方が前面（手前）に表示されます。

z-index: 1
（後ろ）
z-index: 2
（前）
.layer-1 { position: relative; z-index: 1; }
.layer-2 { position: relative; z-index: 2; } /* こちらが前面 */

3レイヤーの重なり

z-index の数値で3段階の前後関係を作れます。

z-index: 1
z-index: 3
z-index: 2

モーダル風のオーバーレイ

モーダルは高い z-index でコンテンツより前に表示します。

コンテンツ（z-index: 1）
オーバーレイ（z-index: 10）
モーダル（z-index: 20）
一番前面
.overlay { position: absolute; z-index: 10; }
.modal   { position: absolute; z-index: 20; } /* 最前面 */
