5. position

要素の配置方法を指定します。

position: static（デフォルト）

通常の文書の流れに従って配置されます。top / left などは無効です。

(デフォルト)

position: relative（自分の元位置を基準にずらす）

元の位置を保ったまま、top / left などで見た目だけずらします。

position: absolute（親要素を基準に配置）

直近の position: relative な親を基準に配置されます。文書の流れから外れます。

position: fixed（画面に固定）

スクロールしても同じ位置に表示されます。ヘッダーや「トップへ戻る」ボタンに使います。

ページ右下の 「▲ TOP」ボタン が fixed の例です。スクロールしても追いかけてきます。

モーダルオーバーレイも position: fixed を使って画面全体を覆います。

.overlay {
  position: fixed;
  inset: 0;          /* top / right / bottom / left を一括で 0 に */
  background: rgba(0, 0, 0, 0.5);
  z-index: 100;
}
