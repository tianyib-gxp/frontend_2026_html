8-2. 非同期 練習

TypeScriptでコールバックを使った非同期処理を書いてみましょう。

問題：遅延メッセージ

  1. delayMessage(message, delay, callback) を作る
  2. delayミリ秒後に callback(message) を呼ぶ
  3. 関数・引数・コールバックに型を付ける
実行

    JavaScriptとの違い

    
// JavaScript
function delayMessage(message, delay, callback) {
  setTimeout(() => {
    callback(message);
  }, delay);
}

// TypeScript
function delayMessage(
  message: string,
  delay: number,
  callback: (msg: string) => void
): void {
  setTimeout(() => {
    callback(message);
  }, delay);
}

    TypeScriptでは、関数の引数として渡す関数（コールバック）にも型を付けます。
    「どんな引数を受け取る関数なのか」を明確にできます。

    学習ポイント

    項目 コード例 説明
    引数の型 message: string 文字列のみ受け取る
    数値の型 delay: number ミリ秒を数値で扱う
    関数型 (msg: string) => void 関数を引数として扱う型
    戻り値 (): void 値を返さない関数
    DOMの型 as HTMLUListElement null対策として型を明示する
