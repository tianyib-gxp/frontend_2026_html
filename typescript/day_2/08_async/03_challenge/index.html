8-3. 非同期 チャレンジ

TypeScriptでコールバックを連鎖させて処理をつなげましょう。

ボーナス問題：模擬データパイプライン

  1. 3つの非同期関数を作る
  2. 引数・戻り値・コールバックに型を付ける
  3. コールバックで順番に処理をつなぐ
実行

    JavaScriptとの違い

    
// JavaScript
function fetchUser(id, callback) { ... }

// TypeScript
function fetchUser(
  id: number,
  callback: (user: User) => void
): void { ... }

    TypeScriptでは、コールバックで受け渡すデータの型も定義します。

    学習ポイント

    項目 コード例 説明
    オブジェクト型 type User = { id: number; name: string } データ構造を定義する
    関数型 (user: User) => void コールバックの型
    非同期処理 setTimeout 時間差で処理を実行する
