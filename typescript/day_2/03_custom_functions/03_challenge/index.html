3-3. 独自の関数 チャレンジ

複数の関数に型を付けて、ミニ買い物カートを作りましょう。

ボーナス問題：ミニ買い物カート

  1. 関数 calculateTotal(prices) を作る
    → 配列 prices の合計金額を return する
  2. 関数 applyDiscount(total, percent) を作る
    → total に percent% の割引を適用した金額を return する
  3. すべての引数・戻り値・変数に型を付ける
  4. 型を付けたあと、以下を試してください
    • applyDiscount("1000", 10);

商品リスト: [1200, 800, 3500, 650]

実行

    JavaScriptとの違い

    
// JavaScript
function calculateTotal(prices) {
  // 処理
}

// TypeScript
function calculateTotal(prices: number[]): number {
  // 処理
}

    TypeScriptでは、配列や数値にも型を付けることができます。
    number[] のように書くことで、「数値の配列」であることを明示します。
    また、: number によって戻り値の型も制限できます。

    学習ポイント

    項目 コード例 説明
    配列の型 number[] 数値の配列であることを示す
    複数引数の型 (total: number, percent: number) それぞれの引数に型を付ける
    戻り値 (): number 関数が数値を返すことを示す
    型チェック applyDiscount("1000", 10) 誤った型はエラーになる
