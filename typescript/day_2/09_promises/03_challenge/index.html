9-3. プロミス チャレンジ

async/await と型を使って、安全な非同期処理を書きましょう。

ボーナス問題：async/await + 型付け

  1. 各関数の戻り値に型を付ける
  2. Promise の型を明示する
  3. login 関数を async/await で実装する
  4. try/catch でエラー処理を行う
  5. エラーの型も考えて記述する
正しい認証情報 間違った認証情報

    JavaScriptとの違い

    
// JavaScript
async function login(username, password) {
  const result = await authenticate(username, password);
}

// TypeScript
async function login(username: string, password: string): Promise<void> {
  const result = await authenticate(username, password);
}

    TypeScriptでは、async関数にも戻り値の型（Promise）を付けます。
    また、APIの返り値の構造も型で明示することで安全に扱えます。

    項目 コード例 説明

    Promiseの型 Promise<User> resolveされる値の型
