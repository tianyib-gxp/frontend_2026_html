9-2. プロミス 練習

Promise チェーンに型を付けて、安全に非同期処理を書いてみましょう。

問題：Promiseチェーン + 型付け

  1. mockFetch(url) は Promise を返す関数です
  2. 戻り値の型を考えて型を付けてください
  3. Promiseチェーンに型を付けながら実装してください
  4. 以下の処理を実装してください:
    • status が "ok" でなければエラーを throw
    • 成功時は「取得成功: ○○件」と表示
    • 失敗時は「エラー発生: ○○」と表示
成功URLで実行 エラーURLで実行

    JavaScriptとの違い

    
// JavaScript
mockFetch(url).then((response) => {
  return response.data;
});

// TypeScript
mockFetch(url).then((response: ApiResponse) => {
  return response.data;
});

    TypeScriptでは、Promiseの中で扱うデータにも型を付けます。
    これにより、response や data の中身が安全に扱えるようになります。

    学習ポイント

    項目 コード例 説明
    Promiseの型 Promise<ApiResponse> Promiseが返す値の型
    引数の型 url: string 関数の入力を明示
    thenの型 (response: ApiResponse) 受け取るデータの型
    エラー処理 catch((err: Error)) エラーの型を扱う
