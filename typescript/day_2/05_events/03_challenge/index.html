5-3. イベント チャレンジ

イベントと型を組み合わせて、フォームの入力チェックを実装しましょう。

ボーナス問題：フォームバリデーション（復習）

  1. フォームの submit イベントを取得する
  2. 入力欄（名前・メール）を取得する
  3. どちらかが空の場合は送信を止める
  4. エラーメッセージを表示する
  5. すべてのDOM・変数に型を付ける

JavaScriptとの違い


// JavaScript
const form = document.getElementById("myForm");

// TypeScript
const form = document.getElementById("myForm") as HTMLFormElement;

TypeScriptでは、フォームや入力欄にも型を付けます。
また、イベントの引数（e）にも型があるため、安全に扱うことができます。

学習ポイント（復習）

項目 コード例 説明
フォームの型 as HTMLFormElement form要素として扱う
inputの型 as HTMLInputElement valueを安全に取得する
イベントの型 (e: Event) イベントオブジェクトの型
null対策 as HTML... 要素取得時の安全性
