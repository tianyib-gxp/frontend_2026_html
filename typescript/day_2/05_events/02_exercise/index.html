5-2. イベント 練習

イベントと型を組み合わせて、安全にクリック処理を実装しましょう。

問題：クリックカウンター（復習）

  1. id が countBtn の要素を取得する
  2. click イベントを追加する
  3. クリックされるたびにカウントを +1 する
  4. id が countDisplay に表示する
  5. すべての変数・DOMに型を付ける
クリック！

クリック数: 0

JavaScriptとの違い


// JavaScript
const btn = document.getElementById("countBtn");

// TypeScript
const btn = document.getElementById("countBtn") as HTMLAnchorElement;

TypeScriptでは、取得したDOM要素に型を付けます。
これにより、どのような要素かを明確にし、安全に操作できます。
