10-2. Promise API 練習

Promise.all() と型を使って、複数の非同期処理を安全に扱いましょう。

問題

  1. 各API関数に戻り値の型を付ける
  2. Promise.all() を使って並行処理を書く
  3. 取得結果に型を付けて表示する
  4. エラー処理にも型を付ける
並行取得を実行

    JavaScriptとの違い

    
// JavaScript
Promise.all([fetchName(), fetchAge(), fetchCity()])
  .then(([name, age, city]) => { ... });

// TypeScript
Promise.all([fetchName(), fetchAge(), fetchCity()])
  .then(([name, age, city]: [string, number, string]) => { ... });

    TypeScriptでは、Promise.all の結果にも型を付けることができます。
    どの順番でどんなデータが返るのかを明確にできます。

    学習ポイント

    項目 コード例 説明
    Promiseの型 Promise<string> 非同期の戻り値の型
    タプル型 [string, number, string] 順序付きの型
    エラー型 err: Error エラーの型
    DOM型 as HTMLUListElement null対策
