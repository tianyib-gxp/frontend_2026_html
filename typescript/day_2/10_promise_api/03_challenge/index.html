10-3. Promise API チャレンジ

Promise.allSettled() と Promise.race() をTypeScriptで使ってみましょう。

問題 1：成功と失敗を全部収集する

  1. Promise.allSettled() を使う
  2. 成功 → 「✅ 成功: 値」
  3. 失敗 → 「❌ 失敗: メッセージ」
  4. 最後に「成功X件 / 失敗Y件」
実行

    問題 2：タイムアウト付きリクエスト

    1. timeout(ms) を作る
    2. Promise.race() を使う
    3. 成功 → 「✅ 結果」
    4. 失敗 → 「⏱️ タイムアウト」
    実行
