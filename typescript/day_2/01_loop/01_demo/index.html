1-1. ループ（for文）TypeScriptデモ

TypeScriptでは型を使って安全にプログラムを書くことができます。
for文と型を組み合わせて、1から5まで表示してみましょう。

for文 + 型

型（number）を使うことで、数値だけを扱う安全なコードになります。
「実行」ボタンを押すと、1から5まで表示されます。

実行
    TypeScriptのサンプルコード
    for (let i: number = 1; i <= 5; i++) {
  addItem(i);
}

    JavaScriptとの違い

    // JavaScript
for (let i = 1; i <= 5; i++) {
  addItem(i);
}

// TypeScript
for (let i: number = 1; i <= 5; i++) {
  addItem(i);
}

    TypeScriptでは : number のように型を付けることで、 扱える値を制限し、安全なコードを書くことができます。
