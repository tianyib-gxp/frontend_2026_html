1-2. ループ（for文）練習

for文を使って、FizzBuzz問題にチャレンジしましょう。

問題：FizzBuzz

  1. 1から20までの数字をループで処理するようになっています
  2. 以下のルールで表示します:
    • 3の倍数 → "Fizz"
    • 5の倍数 → "Buzz"
    • 両方 → "FizzBuzz"
    • それ以外 → 数字
  3. 今のままでも動きますが、関数や変数に「型」をつけてください
  4. 最後に、意図的に間違った値を渡してエラーを確認してください
    • addItem(true); を試す
    • なぜエラーになるか確認する

    TypeScriptでは、型が一致しない場合にエラーが表示されます。
    実際にエラーを確認し、JavaScriptとの違いを理解しましょう。

実行

    学習ポイント

    項目 コード例 説明
    引数の型 text: string | number 文字と数値の両方を扱えるようにする
    戻り値 (): void 何も返さない関数であることを示す
    DOMの型 as HTMLUListElement nullの可能性があるため型を明示する
    変数の型 let i: number 変数に数値型を指定する

    確認ポイント

    以下のコードを試して、TypeScriptのエラーを確認してください。 

    addItem(true);

    JavaScriptでは動作しますが、TypeScriptではエラーになります。
    型によって不正な値を防げることを確認しましょう。
