4-3. 関数の返値 チャレンジ

これまで学んだ関数・型・配列を組み合わせて、処理を完成させましょう。

ボーナス問題：BMI計算ツール（復習）

  1. 関数 calcBMI(weight, height) を作る
    • 体重と身長からBMIを計算して返す
    • 小数点2桁に丸める
  2. 関数 getBMICategory(bmi) を作る
    • BMIに応じて判定を返す
  3. すべての引数・戻り値・変数に型を付ける
  4. 配列（people）にも型を付ける
実行

    JavaScriptとの違い

    
// JavaScript
function calcBMI(weight, height) {
  return weight / (height * height);
}

// TypeScript
function calcBMI(weight: number, height: number): number {
  return weight / (height * height);
}

    TypeScriptでは、複数の引数それぞれに型を付けます。
    また、戻り値にも型を付けることで、関数の役割が明確になります。

    学習ポイント（復習）

    項目 コード例 説明
    複数引数の型 (weight: number, height: number) 引数ごとに型を指定する
    戻り値の型 (): number 関数が返す値の型
    配列の型 { name: string; weight: number; height: number }[] オブジェクト配列の型指定
    変数の型 let i: number ループ変数の型
