4-2. 関数の返値 練習

これまで学んだ「引数・戻り値・型」を使って、関数を完成させましょう。

問題：温度変換関数（復習）

  1. 関数 celsiusToFahrenheit(celsius) を作る
  2. 摂氏（℃）を華氏（℉）に変換して return する
  3. 変換式：℉ = ℃ × 9 / 5 + 32
  4. 引数と戻り値に型を付ける
  5. 配列や変数にも型を付ける
  6. すべての型が正しく付いているか確認する
実行

    JavaScriptとの違い

    
// JavaScript
function celsiusToFahrenheit(celsius) {
  return celsius * 9 / 5 + 32;
}

// TypeScript
function celsiusToFahrenheit(celsius: number): number {
  return celsius * 9 / 5 + 32;
}

    TypeScriptでは、引数と戻り値の両方に型を付けます。
    これにより、関数が「何を受け取り」「何を返すか」が明確になります。

    学習ポイント（復習）

    項目 コード例 説明
    引数の型 celsius: number 関数が受け取る値の型
    戻り値の型 (): number 関数が返す値の型
    配列の型 number[] 数値の配列であることを示す
    変数の型 let i: number ループ変数の型指定
