6-2. オブジェクト 練習

オブジェクトに型を付けて、安全にプロパティやメソッドを扱いましょう。

問題：本のオブジェクト（復習）

  1. book オブジェクトを作る
  2. 各プロパティに型を付ける
  3. メソッド describe にも戻り値の型を付ける
  4. 正しく型が付いているか確認する
実行

    JavaScriptとの違い

    
// JavaScript
const book = {
  title: "JavaScript入門",
  year: 2023
};

// TypeScript
const book: { title: string; year: number } = {
  title: "JavaScript入門",
  year: 2023
};

    TypeScriptでは、オブジェクトのプロパティにも型を付けます。
    これにより、どのようなデータ構造かが明確になります。

    学習ポイント（復習）

    項目 コード例 説明
    オブジェクトの型 { title: string; year: number } プロパティごとに型を定義する
    メソッドの型 (): string 関数の戻り値を指定する
    プロパティアクセス book.title 型に基づいて安全にアクセスできる
