6-3. オブジェクト チャレンジ

TypeScriptではクラスと型を使って、安全にオブジェクトを扱うことができます。

ボーナス問題：社員管理システム（TypeScript）

  1. クラス Employee を作る
  2. 以下のプロパティに型を付ける:
    • name: string
    • department: string
    • salary: number
  3. コンストラクタで値を初期化する
  4. メソッド introduce() を作る（戻り値: string）
    → 「田中さん（営業部）の給与は350000円です」
  5. メソッド getRaise(percent) を作る（戻り値: number）
    → 給与を更新して返す
  6. すべての引数・戻り値・変数に型を付ける
  7. 型チェックとして以下を試す:
    • new Employee(123, "営業", 300000)
実行

    JavaScriptとの違い

    
// JavaScript（コンストラクタ関数）
function Employee(name, department, salary) {
  this.name = name;
}

// TypeScript（クラス + 型）
class Employee {
  name: string;
  department: string;
  salary: number;

  constructor(name: string, department: string, salary: number) {
    this.name = name;
  }
}

    TypeScriptでは、クラスにプロパティの型を定義し、
    コンストラクタやメソッドの引数・戻り値にも型を付けます。
    これにより、オブジェクトの構造が明確になります。

    学習ポイント

    項目 コード例 説明
    クラス class Employee オブジェクトの設計図を作る
    プロパティの型 name: string オブジェクトの中身に型を付ける
    コンストラクタ constructor(...) 初期値を設定する
    メソッドの型 (): string 戻り値の型を明示する
