2-1. 関数（Function）練習

JavaScriptで書いた関数に、TypeScriptで型を付けてみましょう。

問題

  1. 以下の関数に型を付けてください
  2. 引数と戻り値の型を考えてください
  3. 配列や変数にも型を付けてください
  4. 型を付けたあと、以下を試してエラーを確認してください
    • greet(123);
実行

    JavaScriptとの違い

    
        // JavaScript
        function greet(name) {
          return "こんにちは、" + name + "さん！";
        }
        
        // TypeScript
        function greet(name: string): string {
          return "こんにちは、" + name + "さん！";
        }

    TypeScriptでは、関数に「型」を付けることができます。
    引数には name: string のように型を指定し、
    戻り値には : string のように関数が返す値の型を指定します。
    これにより、想定外の値が渡されるのを防ぐことができます。

    学習ポイント

    項目 コード例 説明
    引数の型 name: string 文字列のみ受け取る
    戻り値 (): string 返す値の型を明示する
    配列の型 string[] 配列の中身の型を指定する
    変数の型 let i: number ループ変数に型をつける
