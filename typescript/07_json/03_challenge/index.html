7-3. JSON チャレンジ

JSONデータからカードUIを動的に生成しましょう。

ボーナス問題：社員ディレクトリ

  1. 下記のJSON文字列を解析する
  2. 各社員ごとにカードを作成する
  3. 名前・部署・スキルを表示する
  4. 全スキルの重複なし一覧を表示する
  5. TypeScriptで型を付ける
実行

JavaScriptとの違い


// JavaScript
const employees = JSON.parse(employeesJson);

// TypeScript
type Employee = {
  name: string;
  department: string;
  skills: string[];
};

const employees: Employee[] = JSON.parse(employeesJson);

TypeScriptでは、JSON.parse() の結果に型を付けます。
これにより、どのようなデータ構造なのかが明確になり、 プロパティのミスを防ぐことができます。

学習ポイント

項目 コード例 説明
オブジェクト型 type Employee = { ... } データの構造を定義する
配列の型 Employee[] オブジェクトの配列であることを示す
DOMの型 as HTMLDivElement null対策として型を明示する
配列の操作 string[] スキル一覧のような配列に型を付ける
