7-2. JSON 練習

JSONデータをTypeScriptで扱い、型を付けて安全に処理してみましょう。

問題：商品データをJSONから表示する

  1. JSONの構造に対応する型（Product, Data など）を作る
  2. JSON.parse() の結果に型を付ける
  3. products をループして表示する
  4. 在庫がない場合は「（在庫なし）」と表示する
  5. すべての変数に型を付ける
  6. 型チェックとして以下を試す:
    • data.products[0].price = "100"
実行

    JavaScriptとの違い

    
// JavaScript
const data = JSON.parse(jsonData);

// TypeScript
type Product = {
  name: string;
  price: number;
  inStock: boolean;
};

const data: { products: Product[] } = JSON.parse(jsonData);

    TypeScriptでは、JSONの構造に合わせて型を定義します。
    これにより、存在しないプロパティや型のミスを防ぐことができます。

    学習ポイント

    項目 コード例 説明
    型定義 type Product = {...} オブジェクトの形を定義する
    配列の型 Product[] オブジェクトの配列
    型アサーション as 型 JSON.parse の結果に型を付ける
    boolean型 inStock: boolean true / false を扱う
