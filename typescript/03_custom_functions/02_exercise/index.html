3-2. 独自の関数 練習

複数の引数を持つ関数に型を付けて、条件によって表示を変えてみましょう。

問題：showMessage 関数を作る

  1. showMessage(text, type) を作る
  2. text を表示する
  3. type が "error" の場合 → "❌" をつける
  4. type が "success" の場合 → "✅" をつける
  5. すべての引数・関数・変数に型を付けてください
  6. 型を付けたあと、以下を試してください
    • showMessage("テスト", "warning");
実行

    JavaScriptとの違い

    
// JavaScript
function showMessage(text, type) {
  // 処理
}

// TypeScript
function showMessage(text: string, type: string): void {
  // 処理
}

    TypeScriptでは、複数の引数それぞれに型を付けます。
    また、: void のように戻り値の型も明示します。
    これにより、想定外の値が渡されるのを防ぐことができます。

    学習ポイント

    項目 コード例 説明
    複数引数の型 (text: string, type: string) 引数ごとに型を指定する
    戻り値 (): void 値を返さない関数
    条件分岐 if (type === "error") 値によって処理を変える
    DOMの型 as HTMLUListElement null対策として型を明示
