2-3. 関数 チャレンジ

JavaScriptで作成した関数を、TypeScriptで型を設計しながら書き直してみましょう。

問題：成績判定関数

  1. 関数 calculateGrade を作成してください
  2. スコアに応じて以下の評価を返してください
    • 90以上 → "A"
    • 70以上 → "B"
    • 50以上 → "C"
    • それ以外 → "F"
  3. すべての関数・変数に型を付けてください
  4. 「戻り値の型」を自分で考えてください（重要）
  5. 型を付けたあと、以下を試してください
    • calculateGrade("90");
    • なぜエラーになるか説明してください
実行
