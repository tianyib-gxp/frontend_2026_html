6-1. 配列 デモ

配列の作成・操作・繰り返し・変換メソッドを学びましょう。

配列の基本

基本操作を確認
    配列の作成とアクセス
    const fruits = ['りんご', 'バナナ', 'みかん'];

fruits[0]          // 'りんご'（インデックスは 0 始まり）
fruits.length      // 3
fruits.indexOf('バナナ')  // 1

    要素の追加・削除

    追加・削除を確認
      メソッド説明
      .push(item)末尾に追加
      .pop()末尾を削除して返す
      .unshift(item)先頭に追加
      .shift()先頭を削除して返す
      .splice(i, n)位置 i から n 個削除

      変換メソッド

      変換メソッドを確認
        メソッド説明
        .map(fn)各要素を変換した新しい配列を返す
        .filter(fn)条件に合う要素だけの配列を返す
        .join(sep)配列を文字列に変換
        'a,b'.split(',')文字列を配列に変換
