17-3. Promise API チャレンジ

Promise.allSettled() と Promise.race() を使って応用問題を解きましょう。

問題 1：成功と失敗を全部収集する

5つのAPIリクエストを並行実行し、成功したもの・失敗したもの全てを表示してください。

  1. requests 配列の5つのPromiseを Promise.allSettled() で実行する
  2. 結果を順にループし、成功なら「✅ 成功: 値」、失敗なら「❌ 失敗: メッセージ」と表示する
  3. 最後に「成功X件 / 失敗Y件」と表示する
allSettled を実行

    問題 2：タイムアウト付きリクエスト

    Promise.race() を使い、APIリクエストにタイムアウト（1秒）を設定してください。

    1. fetchData() はランダムな時間（500〜2000ms）で応答するAPIです
    2. timeout(ms) は指定ミリ秒後に拒否するPromiseを返します（実装してください）
    3. Promise.race([fetchData(), timeout(1000)]) を使い、1秒以内なら成功、超えたら「タイムアウト」と表示する
    race + タイムアウトを実行
