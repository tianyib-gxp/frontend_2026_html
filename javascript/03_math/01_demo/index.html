3-1. 数値と演算子 デモ

JavaScriptの算術演算子・代入演算子・比較演算子を学びましょう。

算術演算子

演算結果を表示
    演算子 意味 例 結果
    +加算5 + 38
    -減算5 - 32
    *乗算5 * 315
    /除算5 / 22.5
    %余り5 % 21
    **べき乗2 ** 101024

    代入演算子

    代入演算子を確認
      代入演算子の例
      let x = 10;
x += 5;   // x = x + 5  → 15
x -= 3;   // x = x - 3  → 12
x *= 2;   // x = x * 2  → 24
x /= 4;   // x = x / 4  → 6
x **= 2;  // x = x ** 2 → 36

      比較演算子

      比較演算子を確認
        演算子 意味 例
        ===厳密等価5 === 5 → true
        !==厳密不等価5 !== 3 → true
        <小なり3 < 5 → true
        >大なり5 > 3 → true
        <=以下5 <= 5 → true
        >=以上5 >= 6 → false
