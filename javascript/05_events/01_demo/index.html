5-1. イベント入門 デモ

addEventListener を使って、ユーザー操作に反応するコードを体験しましょう。

クリックイベント

ボタンをクリックすると、背景色がランダムに変わります。
addEventListener("click", ...) でクリックを検知しています。

色を変える

addEventListener のサンプルコード
const btn = document.querySelector("button");

function random(number) {
  return Math.floor(Math.random() * (number + 1));
}

btn.addEventListener("click", () => {
  const r = random(255), g = random(255), b = random(255);
  document.body.style.backgroundColor = `rgb(${r} ${g} ${b})`;
});

キーboardイベント

テキストボックスに入力すると、押したキーがリアルタイムに表示されます。

押したキー: —

keydown イベントのサンプルコード
const input = document.querySelector("input");
input.addEventListener("keydown", (event) => {
  console.log(`"${event.key}" が押されました`);
});
