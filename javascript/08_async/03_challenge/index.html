8-3. 非同期 チャレンジ

コールバックを連鎖させて順番に処理しましょう。

ボーナス問題：模擬データパイプライン

3つの非同期ステップをコールバックで順番に実行する「パイプライン」を実装してください。
各ステップは前のステップの結果を受け取って処理します。

  1. fetchUser(id, callback)：500ms後に { id, name: "田中" } を callback に渡す
  2. fetchOrders(user, callback)：700ms後に { user, orders: ["注文A", "注文B"] } を渡す
  3. calcTotal(data, callback)：300ms後に合計注文数を callback に渡す
  4. 全ステップ完了後、「ユーザー: 田中 / 注文数: 2」を表示する
  5. ログに各ステップの開始時刻（経過ms）も表示する
