8-2. 非同期 練習

コールバックを使った非同期処理を書いてみましょう。

問題：遅延メッセージ

  1. 関数 delayMessage(message, delay, callback) を作る
  2. delay ミリ秒後に callback(message) を呼ぶ（setTimeout を使う）
  3. 「実行」ボタンを押すと、以下の3つのメッセージが順番に表示されること:
    • 500ms後：「ステップ1: データを取得しました」
    • 1000ms後：「ステップ2: データを処理しました」
    • 1500ms後：「ステップ3: 完了しました！」
  4. ボタンを押した瞬間に「処理を開始します...」と表示されることも確認する
