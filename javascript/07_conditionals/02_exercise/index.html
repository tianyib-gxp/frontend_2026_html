7-2. 条件文 練習

天気に応じたメッセージを表示するプログラムを作りましょう。

問題：天気メッセージ表示

  1. ドロップダウンで天気を選択してください
  2. 「表示」ボタンを押すと、天気に応じたメッセージが表示されるようにしてください
  3. if / else if / else を使ってください
  4. 各天気のメッセージ例：
    • sunny（晴れ）→ 「今日は晴れです。お出かけ日和ですね！」
    • cloudy（曇り）→ 「今日は曇りです。念のため傘を持っていきましょう。」
    • rainy（雨）→ 「今日は雨です。傘を忘れずに！」
    • snowy（雪）→ 「今日は雪です。暖かくして出かけましょう。」
    • その他 → 「天気情報が不明です。」
