1-1. JavaScriptとは デモ

HTML・CSS・JavaScriptの3層構造を体験し、JavaScriptがウェブページをどのように動的にするかを確認しましょう。

ウェブの3層構造

ウェブページは3つの技術で成り立っています。
HTML：構造と意味　／　CSS：見た目　／　JavaScript：動作と対話性

技術 役割 例
HTML 構造・意味付け 見出し、段落、リスト
CSS 見た目・スタイル 色、フォント、レイアウト
JavaScript 動作・対話性 ボタンのクリック、DOM操作

JavaScriptで動的なページを作る

下のボタンをクリックすると、JavaScriptがDOMを操作してページのコンテンツを変更します。

まだ名前が入力されていません。

DOM操作のサンプルコード
function updateName() {
  const name = prompt("あなたの名前は？");
  if (name) {
    document.getElementById('greeting').textContent =
      `こんにちは、${name}さん！`;
  }
}

JavaScriptの読み込み方

JavaScriptを HTML に組み込む方法は主に2つあります。

方法 書き方 特徴
内蔵スクリプト <script>...</script> HTML内に直接記述
外部スクリプト <script src="script.js"></script> 別ファイルに分離（推奨）
