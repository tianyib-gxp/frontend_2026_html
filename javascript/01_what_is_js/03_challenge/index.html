1-3. JavaScriptとは チャレンジ

複数のボタンを使って、ページのコンテンツを動的に操作してみましょう。

チャレンジ：インタラクティブなページ

  1. 「テーマ切替」ボタンを押すたびに、#main-box の背景色が白と黒で交互に切り替わるようにしてください
  2. 「テキスト追加」ボタンを押すと、入力欄のテキストを取得して #text-list にリストアイテムとして追加してください
  3. 入力欄が空の場合は何も追加しないようにしてください
  4. 「全て削除」ボタンで #text-list を空にしてください
