4-3. 関数の返値 チャレンジ

複数の返値関数を組み合わせて一つのミニ電卓を作りましょう。

ボーナス問題：BMI計算ツール

  1. 関数 calcBMI(weight, height) を作る
    • weight: 体重（kg）
    • height: 身長（m）
    • BMI = 体重 (kg) ÷ 身長 (m)² を return する（小数点2桁）
  2. 関数 getBMICategory(bmi) を作る
    • 18.5未満 → "低体重"
    • 18.5以上25未満 → "普通体重"
    • 25以上30未満 → "過体重"
    • 30以上 → "肥満"
  3. 下記データのBMIと判定をリスト表示する
