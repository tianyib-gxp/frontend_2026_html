1-1. ループ（for文）デモ

for文を使って、1から5までの数字を順番に表示する仕組みを体験しましょう。

for文の基本

for文は、決まった回数だけ同じ処理を繰り返したいときに使います。
下の「実行」ボタンを押すと、1から5までの数字が順番に表示されます。

実行
    for文のサンプルコード
    for (let i = 1; i <= 5; i++) {
  // 繰り返し処理
}

    他のループとの比較

    種類構文例特徴
    forfor (let i = 0; i < 5; i++)回数が決まっているときに便利
    whilewhile (条件)条件を満たす間ずっと繰り返す
    do...whiledo { ... } while (条件)最低1回は必ず実行される
