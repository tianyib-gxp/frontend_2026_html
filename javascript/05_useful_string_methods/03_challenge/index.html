5-3. 便利な文字列メソッド チャレンジ

駅コードデータをパースして読みやすい形式に変換しましょう。

チャレンジ：駅データのパース

以下の形式のデータが入力欄に入っています。これを解析して表示してください。

データ形式
MAN675847583748sjt567654;Manchester Piccadilly
  1. ; の前にある3文字が駅コードです（MAN）
  2. ; の後にある文字列が駅名です（Manchester Piccadilly）
  3. 駅コードを大文字にして、コード: 駅名 の形式で表示してください
  4. 出力例: MAN: Manchester Piccadilly
