6-3. オブジェクト チャレンジ

コンストラクター関数を使ってオブジェクトを量産しましょう。

ボーナス問題：社員管理システム

  1. コンストラクター関数 Employee(name, department, salary) を作る
  2. メソッド introduce() を追加し、
    「田中さん（営業部）の給与は350000円です」形式の文字列を return する
  3. メソッド getRaise(percent) を追加し、
    給与に percent ％の昇給を反映してから新しい給与額を return する
  4. 3人分の Employee を new で作り、全員の情報と昇給後を表示する
実行
