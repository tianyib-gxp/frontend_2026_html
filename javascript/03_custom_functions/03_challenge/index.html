3-3. 独自の関数 チャレンジ

複数の関数を組み合わせて、ミニ買い物カートを作りましょう。

ボーナス問題：ミニ買い物カート

  1. 関数 calculateTotal(prices) を作る
    　→ 配列 prices の合計金額を return する
  2. 関数 applyDiscount(total, percent) を作る
    　→ total に percent% の割引を適用した金額を return する
  3. 以下の商品リストを使って:
    • 合計を計算する
    • 10%割引を適用する
    • 「合計: ○○円」「割引後: ○○円」の形式で表示する

商品リスト: [1200, 800, 3500, 650]
