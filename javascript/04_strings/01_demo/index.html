4-1. 文字列 デモ

文字列の作成方法、テンプレートリテラル、型変換を学びましょう。

文字列の作成方法

文字列はシングルクォート、ダブルクォート、バッククォートで作成できます。

クォートの種類を表示
    3種類のクォート
    const a = 'シングルクォート';
const b = "ダブルクォート";
const c = `バッククォート（テンプレートリテラル）`;

// 文字列の中にクォートを含める
const d = "It's a pen";           // ダブルの中にシングル
const e = '彼は"こんにちは"と言った'; // シングルの中にダブル
const f = `彼は"hello"と言った`;   // テンプレートの中ならどちらも OK

    テンプレートリテラル

    バッククォートを使うと、${"{変数}"} で変数を文字列に埋め込めます。

    テンプレートリテラルを表示
    テンプレートリテラルの例
    const name = "田中";
const age = 24;

// 連結演算子 (+) での書き方
const msg1 = "こんにちは、" + name + "さん。" + age + "歳ですね。";

// テンプレートリテラルでの書き方（より読みやすい）
const msg2 = `こんにちは、${name}さん。${age}歳ですね。`;

    型変換

    型変換を確認
      変換 方法 例
      文字列 → 数値 Number("42") Number("42") → 42
      数値 → 文字列 String(42) String(42) → "42"
      文字列 + 数値 文字列として連結 "3" + 4 → "34"
