16-1. プロミスの使い方 デモ

Promise の状態・連鎖・エラー処理と、async/await の書き方を学びましょう。

Promiseの基本（then / catch）

Promise は「成功 or 失敗」を表すオブジェクトです。
「成功」ボタンで fulfilled、「失敗」ボタンで rejected を体験できます。

成功パターン 失敗パターン
    Promise チェーンのサンプルコード
    function waitAndResolve(value, ms) {
  return new Promise((resolve, reject) => {
    setTimeout(() => resolve(value), ms);
  });
}

waitAndResolve("データ取得完了", 500)
  .then((result) => {
    console.log(result);   // "データ取得完了"
    return result + " → 加工済み";
  })
  .then((result) => {
    console.log(result);   // "データ取得完了 → 加工済み"
  })
  .catch((err) => {
    console.error("エラー:", err);
  });

    async / await

    async/await は Promise を同期コードのように書ける糖衣構文です。
    「async/await で実行」ボタンを押してみましょう。

    async/await で実行
      async/await のサンプルコード
      async function fetchData() {
  try {
    const result = await waitAndResolve("取得成功！", 500);
    console.log(result);
  } catch (err) {
    console.error("エラー:", err);
  }
}

fetchData();
