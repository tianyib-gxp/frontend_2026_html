10-1. 独自の関数 デモ

自分でゼロから関数を設計し、処理を部品化する考え方を学びましょう。

関数を設計する

関数には「ひとつの役割だけ持たせる」のが基本です。
下の例では摂氏を華氏に変換する専用の関数を作っています。
「実行」ボタンを押すと変換結果が表示されます。

実行
    独自関数のサンプルコード
    function celsiusToFahrenheit(celsius) {
  return celsius * 9 / 5 + 32;
}

// 呼び出し例
const result = celsiusToFahrenheit(0); // → 32

    関数設計のポイント

    ポイント 良い例 避けるべき例
    名前は動詞で calculateTotal() total()
    1つの役割だけ 変換だけを行う関数 変換＋表示を同時に行う関数
    return で値を返す return result; 直接 alert() を使う
