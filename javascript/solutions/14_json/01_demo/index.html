14-1. JSONの操作 デモ

JSON.parse() と JSON.stringify() の使い方を体験しましょう。

JSON.parse() — 文字列 → オブジェクト

サーバーから受け取ったJSON文字列をJavaScriptオブジェクトに変換します。
「実行」ボタンを押して変換の流れを確認しましょう。

実行
    JSON.parse() と JSON.stringify() のサンプル 
    // JSON文字列（サーバーから受け取るイメージ）
const jsonString = '{"name":"田中", "age":28, "skills":["JS","HTML"]}';

// parse: 文字列 → オブジェクト
const obj = JSON.parse(jsonString);
console.log(obj.name);       // "田中"
console.log(obj.skills[0]);  // "JS"

// stringify: オブジェクト → 文字列
const newObj = { city: "Tokyo", active: true };
const newJson = JSON.stringify(newObj);
console.log(newJson); // '{"city":"Tokyo","active":true}'

    JSONの制約まとめ

    項目 JSON
    文字列 必ず ダブルクォート ("key")
    使える型 文字列・数値・真偽値・null・配列・オブジェクト
    使えないもの 関数・undefined・Date
    末尾のカンマ 不可
    コメント 不可
