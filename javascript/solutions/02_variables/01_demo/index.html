2-1. 変数 デモ（解答）

変数はデータを格納するための「箱」です。let・const・データ型を学びましょう。

変数の宣言と初期化

let は再代入可能な変数、const は再代入不可の定数を宣言します。

変数を表示
    変数宣言のサンプルコード
    let myName = "田中";      // 再代入可能
const myAge = 25;         // 定数（再代入不可）
let isStudent = true;     // 論理値

myName = "鈴木";           // OK: let は再代入できる
// myAge = 30;            // エラー: const は再代入不可

    データ型

    JavaScriptには様々なデータ型があります。
    ボタンを押してそれぞれの型を確認してみましょう。

    データ型を表示
      型 例 説明
      数値 (Number) 42, 3.14 整数・小数
      文字列 (String) "hello" テキスト
      論理値 (Boolean) true, false 真偽値
      配列 (Array) [1, 2, 3] リスト
      オブジェクト (Object) {name: "太郎"} 複合データ

      let vs const の使い分け

      キーワード 再代入 用途
      const ❌ 不可 値が変わらない場合（推奨）
      let ✅ 可能 値が後から変わる場合
