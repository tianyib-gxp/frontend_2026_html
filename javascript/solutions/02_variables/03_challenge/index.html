2-3. 変数 チャレンジ（解答）

変数と型変換を使いこなして、簡単な計算アプリを作ってみましょう。

チャレンジ：BMI計算機

  1. 入力欄から身長（cm）と体重（kg）を取得してください
  2. Number() を使って文字列から数値に変換してください
  3. BMI = 体重(kg) ÷ (身長(m) × 身長(m)) を計算してください
  4. 結果を小数点2桁で表示してください（toFixed(2) を使う）
  5. BMIの判定結果（やせ/普通/肥満）も表示してください：
    • BMI < 18.5 → やせ
    • 18.5 ≤ BMI < 25 → 普通
    • 25 ≤ BMI → 肥満
