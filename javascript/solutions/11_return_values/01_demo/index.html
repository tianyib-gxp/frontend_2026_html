11-1. 関数の返値 デモ

関数が計算結果を return で返す仕組みを確認しましょう。

返値のある関数

関数は処理を行うだけでなく、結果の値を呼び出し元に返すことができます。
「実行」ボタンを押すと、入力した数値に対して3つの関数が返値を表示します。


実行
    返値のある関数のサンプルコード
    function squared(num) {
  return num * num;
}

function cubed(num) {
  return num * num * num;
}

function factorial(num) {
  if (num < 0) return undefined;
  if (num === 0) return 1;
  let result = num;
  let x = num - 1;
  while (x > 1) {
    result *= x;
    x--;
  }
  return result;
}

// 返値を変数に受け取る
const n = 4;
console.log(squared(n));   // 16
console.log(cubed(n));     // 64
console.log(factorial(n)); // 24

    返値のポイントまとめ

    ポイント 内容
    return を書く 関数の結果を呼び出し元に渡せる
    return がない場合 関数は undefined を返す
    返値を変数に代入 const result = myFunc(x);
    返値を直接使う console.log(myFunc(x));
