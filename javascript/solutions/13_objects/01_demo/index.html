13-1. オブジェクトの基本 デモ

オブジェクトリテラルの作り方と、プロパティ・メソッドへのアクセス方法を学びましょう。

オブジェクトリテラル

オブジェクトは「名前: 値」のペアの集合です。
「実行」ボタンを押すと、人物オブジェクトの情報が表示されます。

実行
    オブジェクトのサンプルコード
    const person = {
  name: { first: "太郎", last: "山田" },
  age: 28,
  greet() {
    return `こんにちは、${this.name.first}です！`;
  },
};

// ドット記法でアクセス
console.log(person.age);         // 28
console.log(person.name.first);  // 太郎

// ブラケット記法
console.log(person["age"]);      // 28

// メソッドを呼ぶ
console.log(person.greet());     // "こんにちは、太郎です！"

    ドット記法 vs ブラケット記法

    記法 例 使いどき
    ドット記法 person.name プロパティ名が固定のとき（推奨）
    ブラケット記法 person["name"] プロパティ名が変数の場合
