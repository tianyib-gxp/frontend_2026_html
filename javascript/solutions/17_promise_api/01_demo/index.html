17-1. Promise API デモ

4種類のPromise静的メソッドの動作を比較する

メソッド比較表

メソッド いつ解決するか いつ拒否するか
Promise.all() 全て成功したとき どれか1つでも失敗したとき
Promise.any() どれか1つでも成功したとき 全て失敗したとき
Promise.race() 最初に完了したものが成功のとき 最初に完了したものが失敗のとき
Promise.allSettled() 全て完了したとき（失敗も含む） 拒否されない

① Promise.all() — 全成功待ち

Promise.all([p1, p2, p3])
  .then(results => console.log(results))
  .catch(err => console.error(err));
全て成功 1つ失敗

    ② Promise.any() — 最初の成功待ち

    Promise.any([p1, p2, p3])
  .then(first => console.log(first))
  .catch(err => console.error(err)); // AggregateError
    1つ成功 全て失敗

      ③ Promise.race() — 最速優先

      Promise.race([fast, slow])
  .then(winner => console.log("勝者:", winner));
      実行

        ④ Promise.allSettled() — 全結果収集

        Promise.allSettled([p1, p2, p3]).then(results => {
  results.forEach(r => {
    if (r.status === "fulfilled") console.log("成功:", r.value);
    else console.log("失敗:", r.reason);
  });
});
        実行
