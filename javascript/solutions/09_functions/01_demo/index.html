9-1. 関数（Function）デモ

関数を使って、処理をまとめて再利用できる仕組みを体験しましょう。

関数の基本

関数とは、一連の処理をまとめて名前をつけたものです。
下の「実行」ボタンを押すと、名前のリストに挨拶が表示されます。

実行
    関数のサンプルコード
    function greet(name) {
  return "こんにちは、" + name + "さん！";
}

    関数の書き方の比較

    種類 構文例 特徴
    function宣言 function greet(name) { ... } 巻き上げ（hoisting）される
    関数式 const greet = function(name) { ... } 変数に代入する形
    アロー関数 const greet = (name) => { ... } 短く書けるモダンな書き方
